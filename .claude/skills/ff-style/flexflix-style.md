# CORE VISUAL PRINCIPLES

## NEVER USE PURE BLACK

Avoid:
- #000000
- neutral dark grays
- flat black backgrounds

Instead use:
- deep navy
- midnight blue
- indigo
- dark electric blue

Preferred base:
- #060B1E
- #081226
- #0A1330
- #101B44

The interface must always feel luminous and alive.

Even dark sections should contain:
- glow
- gradients
- particles
- subtle color atmosphere

The UI should feel colorful like cinematic thumbnails, NOT monochromatic SaaS.
